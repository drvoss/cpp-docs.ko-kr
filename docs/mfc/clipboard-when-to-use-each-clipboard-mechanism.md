# <a name="clipboard-when-to-use-each-clipboard-mechanism"></a>클립보드: 각 클립보드 메커니즘을 사용하는 경우

클립보드 사용에서 다음 지침을 따릅니다.

- 지금 OLE 클립보드 메커니즘을 사용 하 여 나중에 새로운 기능을 사용할 수 있습니다. 표준 클립보드 API가 유지 되는 동안 OLE 메커니즘은 데이터 전송의 미래입니다.

- Ole 응용 프로그램을 작성 하거나 끌어서 놓기를 비롯 한 OLE 기능을 원하는 경우 ole 클립보드 메커니즘을 사용 합니다.

- Ole 형식을 제공 하는 경우 OLE 클립보드 메커니즘을 사용 합니다.
