# <a name="openmp-in-visual-c"></a>Visual C++의 OpenMP

OpenMP C 및 C++ 응용 프로그램 인터페이스를 사용하면 여러 프로세서를 효과적으로 사용하는 응용 프로그램을 작성할 수 있습니다. Visual C++는 OpenMP 2.0 표준을 지원합니다.
