# <a name="expression-evaluator-error-cxx0047"></a>식 계산기 오류 CXX0047

인수 목록이 함수와 일치 하지 않습니다.

프로그램에 정의 된 동일한 이름의 함수에 대 한 정식 매개 변수 목록과 일치 하지 않는 실제 매개 변수 목록을 사용 하 여 함수를 호출 했습니다.

오버 로드 된 함수는 정확히 일치 하는 매개 변수 또는 개체를 생성할 필요가 없는 일치 하는 경우에만 호출할 수 있습니다.

이 오류는 CAN0047와 동일 합니다.
