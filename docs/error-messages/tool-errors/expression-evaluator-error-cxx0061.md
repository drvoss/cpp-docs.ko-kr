# <a name="expression-evaluator-error-cxx0061"></a>식 계산기 오류 CXX0061

함수 중첩 깊이가 초과 되었습니다.

식에 포함 된 함수 중첩 깊이가 제한 보다 큽니다. 중첩 깊이를 줄이기 위해 식을 수정 합니다.

이 오류는 CAN0061와 동일 합니다.
