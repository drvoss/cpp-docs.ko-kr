# <a name="bscmake-error-bk1514"></a>BSCMAKE 오류 BK1514

모두가. .SBR 파일이 잘렸습니다. 파일 이름에서 찾을 수 없음

업데이트에 대해 지정 된 .sbr 파일이 원래 찾아보기 정보 파일 (.bsc)에 속하지 않습니다. 이 오류를 발생 시킨 .sbr 파일의 이름을 찾으려면 그 앞의 [BK4502](../../error-messages/tool-errors/bscmake-warning-bk4502.md) 경고를 읽으십시오.

### <a name="to-fix-by-checking-the-following-possible-causes"></a>다음과 같은 가능한 원인을 확인하여 수정하려면

1. .Sbr 또는 .bsc에 지정 된 파일 이름이 잘못 되었습니다.

1. .Bsc 파일이 손상 되어 다시 빌드해야 합니다.
