# <a name="expression-evaluator-error-cxx0029"></a>식 계산기 오류 CXX0029

구조체 포인터가 아님

멤버 선택 연산자 ( **->** )가 구조체에 대 한 포인터가 아닌 식에 적용 되었습니다.

전체 식이 올바르게 괄호로 묶지 않았는지 확인 하거나 주소 식을 적절 한 구조체 포인터 형식으로 캐스팅 합니다.

이 오류는 CAN0029와 동일 합니다.
