# <a name="resource-compiler-error-rc2124"></a>리소스 컴파일러 오류 RC2124

빈 메뉴는 허용 되지 않습니다.

**Menu 문에 메뉴** 항목이 정의 되기 전에 **END** 키워드가 표시 됩니다. 리소스 컴파일러는 빈 메뉴를 허용 하지 않습니다. **메뉴** 문 내에 여는 따옴표가 없는지 확인 합니다.
