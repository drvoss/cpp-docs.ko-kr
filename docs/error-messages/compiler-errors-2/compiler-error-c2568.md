# <a name="compiler-error-c2568"></a>컴파일러 오류 C2568

' identifier1 ': 함수 오버 로드를 확인할 수 없습니다.

컴파일러가 호출할 오버 로드 된 함수를 확인할 수 없습니다. 함수에 전달 되는 실제 매개 변수는 오버 로드 된 함수 중 하나에 대 한 정식 매개 변수와 일치 하도록 캐스팅 되어야 하지만, 일치 하는 항목이 없습니다.
