# <a name="requestedit"></a>requestedit

속성이 `OnRequestEdit` 알림을 지원함을 나타냅니다.

## <a name="syntax"></a>구문

```cpp
[requestedit]
```

## <a name="remarks"></a>주의

**Requestedit** C++ 특성에는 [requestedit](/windows/win32/Midl/requestedit) MIDL 특성과 동일한 기능이 있습니다.

## <a name="example"></a>예제

**Requestedit**의 샘플 사용에 대 한 [바인딩](bindable.md) 예제를 참조 하세요.
