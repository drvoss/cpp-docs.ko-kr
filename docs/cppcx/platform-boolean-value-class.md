# <a name="platformboolean-value-class"></a>Platform::Boolean 값 클래스

부울 값을 나타냅니다. 에 해당 하는 **`bool`** 입니다.

## <a name="syntax"></a>구문

```cpp
public value struct Boolean
```

### <a name="members"></a>멤버

부울에는 [Platform::Object Class](../cppcx/platform-object-class.md)에서 파생된 Equals(), GetHashCode() 및 ToString() 메서드와 [Platform::Type Class](../cppcx/platform-type-class.md)에서 파생된 GetTypeCode() 메서드가 있습니다.

### <a name="requirements"></a>요구 사항

**지원 되는 최소 클라이언트:** Windows 8

**지원 되는 최소 서버:** Windows Server 2012

**네임스페이스:** Platform

**메타 데이터:** platform.object
